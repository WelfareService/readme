# 📌 서비스 소개

청년 취약계층은 주거비 부담, 식비 부족, 취업 문제, 심리적 스트레스, 부채, 체납 등  
여러 어려움을 겪고 있지만 **복잡한 복지 정보 검색·서류 탐색** 때문에  
정작 받을 수 있는 혜택을 놓치는 경우가 많습니다.

본 서비스는 복잡한 검색 없이 **자율적인 고민 대화만으로 복지 혜택을 자동 추천**해주는  
능동형 AI 복지 탐지 에이전트입니다.

## 📘 서비스 개요
사용자가 챗봇과 자연스럽게 대화를 나누면 AI가 다음을 자동 수행합니다.

### 1. 대화 속 취약 신호 자동 분석
• 주거불안  
• 결식 위험  
• 고립·은둔  
• 저소득  
• 미취업  
• 심리·정서적 위기  
• 부채  
• 체납  
• 건강위험  
• 돌봄공백  
• (기타 항목들 )

### 2. 복지 정책 & 지자체 데이터 기반 맞춤형 혜택 추천  
대화 내용, 개인 태그, 거주지(산격동) 기반 초개인화 매칭 수행

### 3. GPS 기반 복지센터·지원기관 시각화  
• 복지센터, 보건소, 청년센터 등 지도 마커 표시  
• 클릭 시 기관 정보·거리·업무시간·필요서류·신청방법 제공  

**→ 대화만 하면, 지금 받을 수 있는 개인 맞춤형 복지 혜택을 즉시 확인할 수 있습니다.**

# 🚀 타 서비스와의 차별점

## 1) 지역 기반 · 초개인화된 복지 추천
기존 서비스는 사용자가 '스스로 검색어를 입력'해야만 정보를 얻을 수 있습니다.  
본 서비스는 다음 데이터를 종합해 **AI가 직접 복지를 선별합니다.**

• 나이  
• 거주지(산격동)  
• 개인 태그(저소득·미취업 등)  
• 대화 기반 취약 신호  
• benefits.json 정책 데이터  

→ **사용자는 아무것도 검색하지 않아도 적합한 복지 혜택을 바로 받습니다.**

## 2) 대화형 ‘취약 신호’ 자동 분석 시스템
프롬프트 엔지니어링 기반 AI Match Engine이 대화 속 위험 신호를 감지하고  
**가중치 기반 복지 우선순위**를 실시간 재정렬합니다.

→ 단순 키워드 검색이 아니라, **상황·정서·위험도까지 반영**된 추천

## 3) 사용자 행동 기반 최적화
• 이미 거절한 정책 자동 제외 (Reject-benefit)  
• base_tags(저소득·미취업 등)에 기본 가중치 부여  
• 대화를 오래 이어갈수록 추천 정확도 증가 (Dynamic Priority)

→ **사용자에게 불필요한 정보 없이 필요한 복지만 선별 제공**

## 4) 지도 기반 복지 시각화
• 위치 기반 정책 지도 마커 표시  
• 마커 클릭 시 거리, 기관 정보, 업무시간, 신청 방법 제공  

→ 추천받은 복지의 “위치·접근성·신청 절차”까지 한 번에 확인 가능

# 🔧 향후 개선 및 발전 방향


## 1) 데이터 기반 정책 컨설팅 (Data Business)

대화형 챗봇을 통해 수집되는 청년들의 지역별 복지 수요·위험 신호·선호 정책 데이터는
지자체, 연구기관, 사회복지재단에서 정책 기획·예산 편성·사업 효과 분석의 핵심 근거 자료로 활용할 수 있습니다.

• 지역별 취약 신호 변화(주거·심리·부채·결식 등)  
• 청년들이 실제로 관심을 보인 복지 정책 선택 패턴  
• 거절/선호/미이용 정책 데이터  
• 실시간 위험 증가 구간(동 단위) 시계열 데이터  

이러한 데이터를 기반으로 **정책 인사이트 리포트**를 생성해
지자체·연구기관·복지재단 대상으로 정기 구독(Subscription) 형태의
**데이터 컨설팅 사업으로 확장**이 가능합니다.

---

## 2) 복지 서류 자동 생성 기능

현재 복지 신청 과정은 복잡한 서류 작성, 증빙 자료 준비, 제출 절차 등으로 인해
취약계층 청년에게 높은 장벽이 됩니다.  
대화형 챗봇에서 사용자가 입력한 정보·상황·개인 태그를 기반으로 자동으로 서류를 생성해  
**신청서 PDF 자동 작성 → 제출용 패킷 생성 → 기관 제출 방식 안내**까지 이어지는
원스톱 복지 신청 자동화 기능으로 확장할 수 있습니다.

→ “대화만 하면 신청서가 자동으로 준비되는” AI 기반 복지 Auto-filing 서비스

---

## 3) 위기 청년 실시간 모니터링 대시보드 (B2G SaaS)

AI가 감지한 위험 신호(주거·채무·정서·고립 등)를 **지역·시간대·변화 추세** 기준으로 분석해  
지자체 복지과, 청년센터, 사회복지기관이 바로 활용 가능한  
**실시간 위기 모니터링 대시보드**로 제공할 수 있습니다.

• 동 단위 취약 신호 증가 Heatmap  
• 긴급 위험 키워드 자동 알림  
• 정책 효과성 분석(정책 사용·거절 패턴)  

→ 지역별 위기 상황을 조기에 발견하고, 선제적 개입을 돕는 B2G SaaS 모델로 확장 가능

---

## 4) 기업 CSR 매칭 플랫폼 (B2B)

서비스를 이용하는 청년들의 실제 고충 데이터를 기반으로  
기업·기관의 CSR(사회공헌) 프로그램과 연계할 수 있습니다.

• 편의점/배달 플랫폼의 식비 지원 쿠폰  
• 지역 병원의 건강검진 할인  
• 금융사의 저금리 대출/채무조정 프로그램  
• 프랜차이즈의 청년 월세 지원 캠페인  

기업은 CSR 효과를 데이터 기반으로 검증할 수 있고,  
청년은 현실적인 도움을 즉시 받을 수 있어 **기업–청년–지역 모두에게 이득이 되는 매칭 플랫폼**으로 확장됩니다.

---

## 5) Hyper-local 지역 복지 지도 고도화

현재는 기본 지도 마커 중심의 복지 기관 안내 수준이지만  
향후에는 지역별 복지 자원, 공공기관, NGO, 무료 급식소, 상담센터 등  
**초세분화된 Hyper-local 복지 데이터 플랫폼**으로 확장할 수 있습니다.

• 실시간 혼잡도/대기시간 연동  
• 기관별 서비스 만족도 반영  
• 동 단위 복지 수요 대비 공급 현황 시각화  
• 청년 이동 동선 기반의 개인 맞춤형 루트 추천  

→ 지역 내 모든 복지 자원을 통합한 “지역 기반 복지 내비게이션”으로 고도화 가능

---

## 6) 지속적 RAG 기반 정책 최신화

복지 정책은 지자체·중앙정부·기관마다 변경 주기가 다르고  
서식·자격·지원 범위가 자주 바뀌는 특성이 있습니다.  
이를 자동으로 반영하기 위해

• 크롤링 기반 최신 정책 수집  
• RAG 기반 정책 정보 자동 요약·정제  
• 개정사항 자동 비교 및 업데이트  
• 사용자 상황 변화(취업/퇴사/이사 등)에 따른 재추천  

이 가능해지면 실제 정책 변동성을 실시간으로 반영하는  
**지속 업데이트형 맞춤 복지 추천 엔진**으로 발전합니다.
